# 🟡 MINIONSTORE 3.0 - BOT DE VENDAS DISCORD
Bot profissional completo para vendas automáticas no Discord.

## 🚀 INSTALAÇÃO
1. Edite o arquivo **.env** com suas informações
2. Instale dependências: `npm install`
3. Registre comandos: `npm run deploy`
4. Ligue o bot: `npm start`

## ⚙️ COMANDOS PRINCIPAIS
### ADMIN
- `/addproduto` - Adiciona produtos
- `/criarpainel #canal` - Cria painel da loja
- `/entregar #pedido` - Confirma entrega
- `/giftcard` - Cria giftcards
- `/sorteio` - Cria sorteios

### USUÁRIO
- `/carteira` - Ver saldo
- `/meuspedidos` - Histórico
- `/usargiftcard` - Resgatar giftcard

## 📤 COMO SUBIR NO RENDER
1. Crie conta no render.com
2. New > Web Service > Conecte seu GitHub
3. Build Command: `npm install && npm run deploy`
4. Start Command: `npm start`
5. Cole as variáveis do .env em Environment
6. Deploy!
